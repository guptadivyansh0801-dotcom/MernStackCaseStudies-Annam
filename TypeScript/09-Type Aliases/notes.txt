What I Built:
- Type aliases for OrderStatus, Customer
- Generic Container type
- Function type alias for Logger

Key Learnings:
- type keyword creates alias
- Aliases make code cleaner
- Can alias primitives, objects, functions
- Generics work with type aliases

- Generic syntax <T> was confusing
- Function type syntax arrow vs regular
- When to use alias vs interface
